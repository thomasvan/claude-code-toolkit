# Wabi-Sabi Classification Guide

## Wabi-Sabi Markers (Authentic Imperfections to KEEP)

| Marker Type | Example | Classification |
|-------------|---------|----------------|
| Natural typo | "orchistrates" | WABI-SABI (KEEP) |
| Transposed letters | "improvments" | WABI-SABI (KEEP) |
| Run-on enthusiasm | Long comma-chain sentence (60+ words) | WABI-SABI (KEEP) |
| Fragment punch | "That's it." | WABI-SABI (KEEP) |
| Self-correction | "okay not entirely" | WABI-SABI (KEEP) |
| Trailing thought | Sentence that doesn't quite... | WABI-SABI (KEEP) |
| Casual contraction | "don't", "it's", "aren't" | WABI-SABI (KEEP) |
| Repeated emphasis | "All of it was leading somewhere. All of it mattered." | WABI-SABI (KEEP) |
| Parenthetical tangent | Mid-sentence aside in parentheses | WABI-SABI (KEEP) |
| "And" sentence start | "And then it all clicked." | WABI-SABI (KEEP) |

## Actual Violations (Errors to FIX)

| Violation | Example | Classification |
|-----------|---------|----------------|
| Em-dash | "The system -- complex -- fails" | ERROR (FIX) |
| AI construct | "It's not X, it's Y" | ERROR (FIX) |
| Corporate jargon | "leverage synergies" | ERROR (FIX) |

| Accusatory | "You probably haven't..." | ERROR (FIX) |
| Balanced parallel | "Not only X, but also Y" | ERROR (FIX) |
| Smooth transition | "Furthermore", "Moreover" | ERROR (FIX) |

## Classification Decision Tree

```
Is the pattern found in the writer's authentic samples?
    |
    +-- YES -> WABI-SABI (KEEP)
    |
    +-- NO -> Is it a natural speech pattern?
                |
                +-- YES -> WARNING (REVIEW)
                |
                +-- NO -> Is it on the banned list?
                          |
                          +-- YES -> ERROR (FIX)
                          |
                          +-- NO -> WARNING (REVIEW)
```

## The Suspiciously Perfect Problem

Zero wabi-sabi markers is itself a red flag. Authentic human writing always
contains some imperfections. If an article scores 95+ with no markers found,
that is more suspicious than an article scoring 75 with several markers.

| Wabi-Sabi Count | Signal |
|-----------------|--------|
| 0 markers | Suspicious - likely AI-generated or over-edited |
| 1-2 markers | Minimal - possibly lightly edited AI |
| 3-5 markers | Natural - typical human writing |
| 6+ markers | Very natural - unedited or lightly edited |
